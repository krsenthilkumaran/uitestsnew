Step1: Install NodeJS
		Check NodeJS version using command  --> npm -version

Step2: Update NPM using command  --> npm install -g npm

Step3: Install Protractor using the command --> npm install protractor / npm install -g protractor

Step4: Install jasmine-allure-reporter package for report generation using command --> npm i jasmine-allure-reporter  / npm i -g jasmine-allure-reporter

Step5: Install xls to Json converter package using command --> npm install xls-to-json / npm install -g xls-to-json

Step6: If you want to generate HTML report then we need to install Jasmin reporter using command  npm install --save-dev jasmine-reporters@^2.0.0 / npm install –g jasmine-reporters@^2.0.0

Step7. Install jasmine2-protractor-utils using command -- npm i jasmine2-protractor-utils

Step8: Install Jasmine data Provider to use Loops using command --> npm i jasmine-data-provider

Step9: Download Mavan - apache-maven-3.5.3 - and config the environmental variables for mavan

Step10: Place the Mavan ( pom.xml ) in to ../node_modules\jasmine-allure-reporter - folder to build the Allure report

step11: Update webdriver using command --> webdriver-manager update

Step12: Start Webdriver using command -->  webdriver-manager start

Step13: Create below listed folders to configure the Framework

Folders:

Data - contains Test data file (Testdata.json)
Pages - contains global functions which can be used with in code (Page objects)
Specs - contains Test Scripts

Files:

Config.json - Base file to trigger the process

Step8: Install default packages using command --> npm install

Step9: Install Webstorm - JS editor

Optional:


Step15: MAven report generation command - mvn site -Dallure.results_pattern=allure-results

Step 16 : Install Protractor hot keys using command - npm install -g protractor-hotkeys


Workflow Process

1. Login as a maker
2. Creating a draft from Master
3. Search for Channels in master and creation of Channels (Positive & NEgative Flow)
4. Search for Currency in master and creation of Currency (Positive & NEgative Flow)
5. Search for Current Employee Status in master and creation of  Current Employee Status (Positive & NEgative Flow)
