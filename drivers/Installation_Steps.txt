Mandatory Steps
---------------

Step1: Minimum requirment of Java version is 1.8

Step2: Install NodeJS
		Check NodeJS version using command  --> npm -version
		
Step3: Verify the Environment Variables are been set in PATH

Step4: Install default packages using command --> npm install		
		
Step5: Update NPM using command  --> npm install -g npm

Step6: Install Protractor using the command -->  npm install -g protractor

Step7. Install jasmine2-protractor-utils using command -- npm install -g jasmine2-protractor-utils

Step8: Install Jasmine data Provider to use Loops using command --> npm install -g jasmine-data-provider

Step9 : Install Protractor hot keys using command - npm install -g protractor-hotkeys

Step10: Install jasmine-allure-reporter package for report generation using command --> npm i jasmine-allure-reporter  / npm i -g jasmine-allure-reporter

step11: Update webdriver using command --> webdriver-manager update

Step12: Start Webdriver using command -->  webdriver-manager start

Report Generation with Maven
----------------------------

Maven report generation command - mvn site -Dallure.results_pattern=allure-results



Additional Packages
--------------------

Step13: Install xls to Json converter package using command --> npm install xls-to-json / npm install -g xls-to-json

Step14: Download Mavan - apache-maven-3.5.3 - and config the environmental variables for mavan

Step15: Place the Mavan ( pom.xml ) in to ../node_modules/jasmine-allure-reporter - folder to build the Allure report

Step16: Create below listed folders under the parent folder to configure the Framework

Data - contains Test data file (Testdata.json)
Pages - contains global functions which can be used with in code (Page objects)
Specs - contains Test Scripts

Tigger File:
------------

Config.json - Base file to trigger the process

Optional installation - JS Editor
---------------------------------

Step17: Install Webstorm - JS editor
